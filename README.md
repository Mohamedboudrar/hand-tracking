# Hand Tracking with MediaPipe & OpenCV

## Installation

### Windows

1. Create virtual environment

    python -m venv venv

2. Activate virtual environment

    venv\Scripts\activate

3. Install dependencies

    pip install -r requirements.txt

4. Run application

    python main.py


### macOS / Linux

1. Create virtual environment

    python3 -m venv venv

2. Activate virtual environment

    source venv/bin/activate

3. Install dependencies

    pip install -r requirements.txt

4. Run application

    python main.py
